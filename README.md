# metalsmith-jekyll [![Build Status](https://secure.travis-ci.org/danielhusar/metalsmith-jekyll.svg?branch=master)](http://travis-ci.org/danielhusar/metalsmith-jekyll)

> Static jekyll-like blog generator.

## Features:

  - drafts
  - markdown
  - excerpts
  - pagination
  - rss
  - tags
  - sass
  - autoprefixer
  - uglifyjs
  - imagemin
